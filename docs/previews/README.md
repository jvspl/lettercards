# docs/previews/

This folder holds before/after screenshots for pull requests that change card visuals.

These images are **not part of the project output** — they exist only so PR descriptions can reference them via raw GitHub URLs before the branch is merged (GitHub doesn't render relative image paths in PR bodies until after merge).

Files here follow the naming convention `issue-{N}-before.png` / `issue-{N}-after.png` and are committed to the feature branch alongside the change they illustrate. They can be cleaned up after merge.
